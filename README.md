# vue-movie-app

Implementing Udemy Course "Vue.js: Learning the basics by building a movie web app"

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
